Ход работы: 
- Графики зависимости кол-ва эпох от ошибки обучения на каждой логической операции:

 ![33](https://user-images.githubusercontent.com/114075427/204216497-8210a464-0d67-4c71-b739-92c2b51c69a8.png)
- Графики totalError(AND) и totalError(NAND) совпадают. Для операций: OR, AND и NAND необходимо примерно одиноковое кол-во эпох для приближения totalerror к нулю. Для XOR все сложнее.
